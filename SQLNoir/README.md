# SQLNoir Writeups 

This folder contains my writeups for the **SQLNoir** cases. SQLNoir is an interactive SQL investigation game where you play the role of a detective solving crimes using SQL queries.

Each case presents a different scenario where a crime has taken place, and the goal is to analyze the database, gather clues, and identify the culprit. The investigation usually involves querying tables such as crime scenes, witnesses, suspects, and interview transcripts.

Throughout these cases, I practice using SQL to:

- Retrieve relevant information from the database
- Filter suspects based on clues from witnesses
- Analyze evidence from different tables
- Confirm the culprit through interview transcripts

Each writeup documents the investigation process step by step, including the SQL queries used and the reasoning behind them.
